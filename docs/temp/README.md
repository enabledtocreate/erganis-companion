# Temporary documentation (Companion)

Staging area for mobile app, Public API consumption, and Companion feature notes **before** APM-managed docs exist for this repo.

## Use this folder for

- Public API integration and auth (JWT) notes
- MapLibre mobile spikes

- Hand-editing final managed docs in-repo (use APM when available)
- Secrets or `.env` values

## Cleanup

When a topic is promoted to APM or the product plan, remove or archive the scratch file here.
